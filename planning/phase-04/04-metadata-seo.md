## 🔍 SEO STRATEGY

### Keyword Integration Principles

**1. Natural Integration**
- Keywords should flow naturally in content
- Avoid keyword stuffing
- Focus on reader experience first

**2. Keyword Placement Priority**
- H1 (page title)
- First paragraph
- H2 subheadings
- Meta title and description
- Image alt text
- URL slug

**3. Keyword Variations**
- Use synonyms and related terms
- Include long-tail variations
- Consider user search intent

### Content SEO Best Practices

**Heading Structure:**
- One H1 per page (includes primary keyword)
- Multiple H2s (include secondary keywords)
- H3s for subsections
- Never skip heading levels

**Content Length:**
- Homepage: 500-800 words
- Service pages: 1000-1500 words
- Product pages: 300-500 words
- Educational pages: 1500-2500 words
- Policy pages: As needed

**Internal Linking:**
- 3-5 internal links minimum per page
- Descriptive anchor text
- Link to pillar pages
- No orphan pages

**Image Optimization:**
- Descriptive file names (yoni-steam-blend-moon-cycle.jpg)
- Alt text with keywords (natural, not stuffed)
- Compressed file sizes (< 500KB)
- WebP format with JPG fallback

## 📊 METADATA REQUIREMENTS

### Required Metadata for Every Page

**1. Meta Title**
- **Length:** 50-60 characters (optimal)
- **Format:** [Page Topic] - Sacred Portal Wellness
- **Include:** Primary keyword
- **Example:** "Holistic Wellness Coaching - Sacred Portal Wellness"

**2. Meta Description**
- **Length:** 150-155 characters (optimal)
- **Include:** Primary keyword, value proposition, CTA
- **Compelling:** Encourage click-through
- **Example:** "12-week holistic wellness coaching for women. Balance hormones, support cycles, boost energy. Led by Amber, RN, BSN, HC-NC. Free discovery call."

**3. Canonical URL**
- **Format:** https://sacredportalwellness.com/[page-slug]
- **Purpose:** Prevent duplicate content issues
- **Required:** Every page

**4. Open Graph Tags**
- **og:title** - Social sharing title (can differ from meta title)
- **og:description** - Social sharing description
- **og:image** - Social sharing image (1200x630px)
- **og:url** - Canonical URL
- **og:type** - Page type (website, article, product)

**5. Twitter Card Tags**
- **twitter:card** - summary_large_image
- **twitter:title** - Same as og:title
- **twitter:description** - Same as og:description
- **twitter:image** - Same as og:image

## 📱 OPEN GRAPH & SOCIAL SHARING

### Image Specifications

**OG Image Dimensions:**
- **Size:** 1200 x 630 pixels
- **Format:** JPG or PNG
- **File Size:** < 300KB
- **Aspect Ratio:** 1.91:1

**Image Content:**
- Brand colors (Forest Green, Moss Green, Cream)
- Logo or brand mark
- Page title or key message
- Minimal text (readable on mobile)
- High contrast for readability

### OG Image Requirements by Page Type

**Homepage:**
- Brand hero image
- Logo + tagline
- Mystical/earthy aesthetic

**Coaching:**
- Coaching program branding
- "The Sacred Portal Path" text
- Transformation imagery

**About:**
- Professional photo of Amber
- Credentials: RN, BSN, HC-NC
- Warm, approachable

**Educational (Yoni Steaming):**
- Educational graphic
- Key benefits or how-to visual
- Informative, trustworthy

**Shop:**
- Product collection image
- Multiple products displayed
- "Shop" or category name

**Product Pages:**
- Primary product image
- Product name
- Price (optional)

## 🏷️ SCHEMA MARKUP

### Recommended Schema Types

**1. Organization Schema (Homepage)**
```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Sacred Portal Wellness",
  "url": "https://sacredportalwellness.com",
  "logo": "https://sacredportalwellness.com/images/logo.png",
  "description": "Holistic wellness coaching and yoni steaming products for women",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "San Diego",
    "addressRegion": "CA",
    "addressCountry": "US"
  },
  "sameAs": [
    "https://www.instagram.com/saacredportal"
  ],
  "contactPoint": {
    "@type": "ContactPoint",
    "email": "info@sacredportalwellness.com",
    "contactType": "Customer Service"
  }
}
```

**2. Person Schema (About Page)**
```json
{
  "@context": "https://schema.org",
  "@type": "Person",
  "name": "Amber",
  "jobTitle": "Holistic Nurse Coach",
  "description": "RN, BSN, HC-NC with 10 years healthcare experience",
  "url": "https://sacredportalwellness.com/about",
  "image": "https://sacredportalwellness.com/images/amber.jpg",
  "sameAs": [
    "https://www.instagram.com/saacredportal"
  ]
}
```

**3. Product Schema (Product Pages)**
```json
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "[Product Name]",
  "description": "[Product Description]",
  "image": "[Product Image URL]",
  "brand": {
    "@type": "Brand",
    "name": "Sacred Portal Wellness"
  },
  "offers": {
    "@type": "Offer",
    "price": "[Price]",
    "priceCurrency": "USD",
    "availability": "https://schema.org/InStock",
    "url": "[Product URL]"
  }
}
```

**4. Service Schema (Coaching Page)**
```json
{
  "@context": "https://schema.org",
  "@type": "Service",
  "serviceType": "Holistic Wellness Coaching",
  "provider": {
    "@type": "Organization",
    "name": "Sacred Portal Wellness"
  },
  "description": "12-week personalized holistic wellness coaching program for women",
  "areaServed": {
    "@type": "Place",
    "name": "Nationwide (Virtual)"
  }
}
```

**5. FAQ Schema (FAQ Page)**
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "[Question Text]",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "[Answer Text]"
      }
    }
  ]
}
```
