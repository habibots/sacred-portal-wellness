# Phase 4: Content Quality Control Checklist

**Project:** Sacred Portal Wellness Website  
**Document:** Content QC Checklist  
**Purpose:** Ensure all content meets quality standards before approval  
**Last Updated:** March 12, 2026

---

## 📋 MASTER CHECKLIST

Use this checklist for every page before marking it "Approved - Ready for Development"

---

## ✅ CONTENT COMPLETENESS

### Required Sections

- [ ] Page title (H1) present
- [ ] All required sections from content brief included
- [ ] Hero/intro section (if applicable)
- [ ] Body content complete
- [ ] Primary CTA defined and placed
- [ ] Secondary CTA (if applicable)
- [ ] Trust signals included (credentials, testimonials, etc.)
- [ ] Disclaimer (if required for page type)

### Content Quality

- [ ] Content matches purpose defined in brief
- [ ] Target audience appropriately addressed
- [ ] All information is accurate and factual
- [ ] No placeholder text remaining ("Lorem ipsum", "TBD", etc.)
- [ ] Content flows logically from section to section
- [ ] Transitions between sections are smooth

---

## 📏 CHARACTER LIMITS

### Verify Limits

- [ ] H1 (Page Title): ≤ 60 characters
- [ ] Meta Title: ≤ 60 characters
- [ ] Meta Description: ≤ 155 characters
- [ ] Hero/Intro Section: ≤ 200 characters (if applicable)
- [ ] Button Text: ≤ 25 characters
- [ ] Card Titles: ≤ 50 characters (if applicable)
- [ ] Card Descriptions: ≤ 120 characters (if applicable)
- [ ] Product Summary: ≤ 150 characters (if applicable)

### Component-Specific Limits

- [ ] All component content within Phase 3 design system limits
- [ ] No text overflow in constrained components
- [ ] Mobile character limits considered

---

## 🎨 BRAND VOICE

### Tone Attributes

- [ ] Warm but professional
- [ ] Educational without being preachy
- [ ] Empowering
- [ ] Authentic
- [ ] Earthy, grounded, feminine

### Voice Consistency

- [ ] Consistent with Phase 1 brand guidelines
- [ ] Matches existing approved content
- [ ] Appropriate for target audience (women 20-40)
- [ ] No overly clinical language
- [ ] No condescending tone
- [ ] No vague mystical claims without grounding

---

## 🔍 SEO OPTIMIZATION

### Keyword Integration

- [ ] Primary keyword in H1
- [ ] Primary keyword in first paragraph
- [ ] Keywords in at least one H2
- [ ] Keywords integrated naturally (no stuffing)
- [ ] Keyword variations used appropriately
- [ ] Long-tail keywords included where relevant

### Metadata

- [ ] Meta title includes primary keyword
- [ ] Meta description includes primary keyword
- [ ] Meta description includes value proposition
- [ ] Meta description includes CTA or benefit
- [ ] Canonical URL defined
- [ ] OG tags complete (title, description, image, URL, type)
- [ ] Twitter card tags complete

### Content Structure

- [ ] Proper heading hierarchy (H1 → H2 → H3, no skipping)
- [ ] Descriptive, keyword-rich headings
- [ ] Scannable content (short paragraphs, bullet points)
- [ ] Content length appropriate for page type

---

## 🔗 INTERNAL LINKING

### Link Quantity

- [ ] Minimum 3 contextual internal links
- [ ] Maximum appropriate for content length
- [ ] Links to pillar pages where relevant
- [ ] Links to conversion pages (Shop, Coaching)

### Link Quality

- [ ] Descriptive anchor text (no "click here")
- [ ] Links are contextually relevant
- [ ] Links add value for the reader
- [ ] No broken links
- [ ] Links open in same tab (internal) or new tab (external)

### Navigation

- [ ] Page included in appropriate navigation menu
- [ ] Breadcrumbs defined (if applicable)
- [ ] Footer links appropriate
- [ ] No orphan pages (page is linked from at least 3 other pages)

---

## ♿ ACCESSIBILITY

### Content Accessibility

- [ ] Reading level: 8th grade or below
- [ ] Short paragraphs (2-4 sentences max)
- [ ] Bullet points used for lists
- [ ] Clear, descriptive headings
- [ ] Active voice used predominantly
- [ ] Simple, clear language (no unnecessary jargon)

### Link Accessibility

- [ ] Descriptive link text
- [ ] Link purpose clear from context
- [ ] No "click here" or "read more" without context

### Image Accessibility (noted for Phase 5)

- [ ] Image placements noted
- [ ] Alt text descriptions provided
- [ ] Decorative vs. informative images identified

---

## 🏥 COMPLIANCE & LEGAL

### Health Claims

- [ ] No absolute medical claims ("cures", "treats", "prevents")
- [ ] Evidence-based language ("may help", "supports", "promotes")
- [ ] No guarantees of specific results
- [ ] Appropriate disclaimers present

### Credentials

- [ ] RN, BSN, HC-NC displayed accurately
- [ ] Credentials explained where first mentioned
- [ ] No misrepresentation of qualifications
- [ ] Scope of practice respected

### Disclaimers

- [ ] Health disclaimer on coaching page
- [ ] Product disclaimer on product pages
- [ ] "Not medical advice" where appropriate
- [ ] "Consult healthcare provider" where appropriate

### Policy Compliance

- [ ] Return policy mentioned on product pages
- [ ] Shipping policy mentioned on shop pages
- [ ] Privacy policy linked in forms
- [ ] All legal requirements met for page type

---

## 📱 TECHNICAL REQUIREMENTS

### File Format

- [ ] Markdown format (.md)
- [ ] Proper Markdown syntax
- [ ] Headings formatted correctly (# H1, ## H2, ### H3)
- [ ] Lists formatted correctly (- or 1.)
- [ ] Links formatted correctly ([text](url))
- [ ] No formatting errors

### File Organization

- [ ] Saved to correct folder (`content/pages/`)
- [ ] File named correctly ([page-slug].md)
- [ ] Version control maintained (if applicable)
- [ ] Metadata frontmatter included (if using)

### Content Structure

- [ ] Clear section breaks
- [ ] Logical content flow
- [ ] CTA placements noted
- [ ] Component placements noted
- [ ] Image placements noted

---

## 🎯 CALLS-TO-ACTION

### CTA Presence

- [ ] Primary CTA clearly defined
- [ ] Primary CTA placement strategic
- [ ] Secondary CTA (if applicable)
- [ ] CTAs appropriate for page purpose

### CTA Quality

- [ ] CTA text is action-oriented
- [ ] CTA text is specific (not generic)
- [ ] CTA text ≤ 25 characters
- [ ] CTA destination is clear
- [ ] CTA value proposition is clear

### CTA Examples

**Good:**
- ✅ "Book Your Free Discovery Call"
- ✅ "Shop Yoni Steam Blends"
- ✅ "Read My Story"

**Poor:**
- ❌ "Click Here"
- ❌ "Learn More"
- ❌ "Submit"

- [ ] Specialties mentioned where relevant
- [ ] Professional photo noted (if applicable)

## 📝 GRAMMAR & STYLE

### Grammar

- [ ] No spelling errors
- [ ] No grammatical errors
- [ ] Proper punctuation
- [ ] Consistent tense
- [ ] Subject-verb agreement

### Style

- [ ] Consistent capitalization
- [ ] Consistent formatting
- [ ] Proper use of em dashes, hyphens
- [ ] Numbers formatted consistently (spell out or numerals)
- [ ] Dates formatted consistently

### Readability

- [ ] Sentences vary in length
- [ ] Paragraphs vary in length
- [ ] No run-on sentences
- [ ] No sentence fragments (unless intentional for style)
- [ ] Content is engaging and readable

## 🚨 RED FLAGS

### Stop and Fix If You See:

- ❌ "Cures", "treats", "prevents" (medical claims)
- ❌ "Guaranteed results"
- ❌ "FDA-approved" (unless true)
- ❌ "Medical treatment" (coaching is not medical)
- ❌ "Click here" or "read more" (poor link text)
- ❌ Lorem ipsum or placeholder text
- ❌ TBD or [INSERT X]
- ❌ Broken links
- ❌ Character limits exceeded
- ❌ Missing required sections
- ❌ Inconsistent brand voice
- ❌ Spelling or grammar errors
