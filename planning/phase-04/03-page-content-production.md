**For each page:**
- [ ] Review Phase 1 existing content
- [ ] Review Phase 3 design specifications
- [ ] Identify content gaps
- [ ] Note character limits
- [ ] List required CTAs

### Step 2: Create Content Brief

**Document:**
- Page purpose
- Target audience
- Primary keyword
- Required sections
- Character limits
- Internal links needed

### Step 3: Write First Draft

**Follow:**
- Brand voice guidelines (Phase 1)
- Content model (02-content-model.md)
- Character limits
- SEO requirements

### Step 4: Self-Review

**Check:**
- [ ] All required sections present
- [ ] Character limits respected
- [ ] Keywords integrated naturally
- [ ] CTAs clear and compelling
- [ ] Internal links added
- [ ] Brand voice consistent
- [ ] No unsupported health claims

### Step 5: Save to Content Folder

**File naming:**
```
content/pages/[page-slug].md
```

**Example:**
```
content/pages/homepage.md
content/pages/coaching.md
content/pages/about.md
```

### Step 6: Track Status

**Update:** `PHASE-04-STATUS.md`

**Status options:**
- Draft
- In Review
- Approved
- Ready for Development
