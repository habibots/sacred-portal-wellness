## 📝 How to Add New Testimonials

1. Get written approval from client
2. Copy `testimonial-template.md`
3. Fill in all sections
4. Create 3 versions (short, medium, full)
5. Review for compliance (no unverifiable medical claims)
6. Save to `approved/` folder with naming format: `YYYY-MM-DD_initials-age_testimonial.md`

---

## ⚠️ Compliance Guidelines

**Always check for:**
- No unverifiable medical claims
- No claims about curing diseases
- Focus on experience, not medical outcomes
- Get written approval before using
- Include disclaimer on website

**Safe to include:**
- Personal experience
- Emotional/mental benefits
- Process and support quality
- General wellness improvements
