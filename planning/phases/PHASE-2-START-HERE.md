## ⚠️ COMMON QUESTIONS

### Q: Can I use my existing email instead of Google Workspace?
**A:** You can, but professional email is strongly recommended for:
- Credibility (business@yoursite.com vs gmail.com)
- Deliverability (better inbox placement)
- Branding (matches your website)
- Separation (business vs personal)

### Q: Do I need to provide my home address?
**A:** Only for Google Business Profile verification (postcard). Your address:
- Is NOT shown publicly
- Is only for verification
- Can be removed after verification
- Service area is shown instead

### Q: What if I already have a Square account?
**A:** Perfect! You'll just:
- Create a new application in Square Developer
- Generate API credentials
- Use your existing product catalog

### Q: Can I start Phase 3 while waiting for Google verification?
**A:** Yes! Phase 3 (design) is independent and can start immediately.
