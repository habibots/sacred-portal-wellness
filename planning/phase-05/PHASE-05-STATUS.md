## 📝 Notes & Decisions

### Technical Decisions
- **Framework:** Next.js 14+ with App Router
- **Styling:** Tailwind CSS with design tokens
- **Commerce:** Square APIs (Catalog, Orders, Payments)
- **Hosting:** Vercel
- **State Management:** React Context for cart
- **Validation:** Zod schemas

### Important Reminders
- Never expose Square access tokens client-side
- Always validate server-side
- Use idempotency keys for payments
- Test with Square sandbox before production
